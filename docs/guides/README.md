# Guides - Руководства для разработчиков

Руководства по разработке, архитектуре и лучшим практикам проекта.

## Руководства

- **[Architecture](architecture.md)** - Архитектура системы, слои, взаимодействие компонентов
- **[Development](development.md)** - Руководство по разработке, установка, запуск
- **[Testing](testing.md)** - Тестирование, запуск тестов, best practices
- **[Logging](logging.md)** - Логирование, структурированные логи, мониторинг
- **[Error Handling](error-handling.md)** - Обработка ошибок, исключения, best practices
- **[Migration](migration.md)** - Руководство по миграции на новую структуру
- **[Troubleshooting](troubleshooting.md)** - Кейсы и решения проблем
- **[Project Structure](project-structure.md)** - Детальное описание структуры проекта

## Быстрый доступ

### Для новых разработчиков
1. Начните с [Development](development.md) - установка и запуск
2. Изучите [Architecture](architecture.md) - понимание структуры
3. Прочитайте [Error Handling](error-handling.md) - обработка ошибок
4. При проблемах см. [Troubleshooting](troubleshooting.md) - кейсы и решения

### Для опытных разработчиков
- [Testing](testing.md) - как писать и запускать тесты
- [Logging](logging.md) - структурированное логирование
