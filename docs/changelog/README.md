# Changelog - История изменений

Эта папка содержит консолидированную историю важных изменений в проекте.

## Формат

Каждый файл описывает одно значимое изменение:
- **Дата** - когда было выполнено
- **Статус** - завершено/в процессе
- **Проблема** - что было исправлено
- **Решение** - как было исправлено
- **История изменений** - ссылки на связанные документы

## Файлы

## Добавление новых записей

При добавлении новой записи:
1. Создайте файл с понятным названием: `feature-name.md` или `bugfix-name.md`
2. Используйте формат из существующих файлов
3. Обновите этот README
